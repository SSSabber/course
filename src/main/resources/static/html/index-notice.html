关于2020级新生英语水平测试及各类试点班选拔等事宜的通知

亲爱的2020级新同学：

欢迎你来到武汉理工大学学习。为帮助你顺利开始大学生活，特将入学前后本科教学相关事宜告知你，请你利用暑假做好相应准备，更好地迎接大学学习和生活。

一、英语水平测试

为了更好地组织《大学英语》课程教学，学校将对2020级新生进行英语水平测试，考试成绩将作为大学英语分级教学的依据及部分试点班选拔的依据之一。

（一）考试时间：2020年9月13日晚上19:00-21:00。

（二）考生范围：除以下学生，其他所有2020年入校的普通本科学生均应参加。

不参加考试学生：外国语学院学生,国际教育学院学生、艾克斯马赛学院学生、艺术类专业（设计学类、动画专业）学生、文化课单独考试招生录取的高水平运动队运动员、高考时第一外语非英语的小语种学生。

（三）因故无法参加考试的学生，由学校指定相应普通班学习。

（二）试点班简介及选拔办法

各试点班简介、选拔办法、咨询电话等信息详见以下网站：

1.教务处网站http://jwc.whut.edu.cn/ 2.本科招生网https://zs.whut.edu.cn/

（三）试点班选拔部分科目考试时间

各试点班选拔考试科目见《试点班简介及选拔办法》，其中考试科目为英语和数学的，考试时间如下：

1. 英语：使用英语水平测试成绩，考试时间 9月13日晚上19:00-21:00；

2. 数学：考试时间9月18日晚上19:00-21:00。

（四）报名方式及时间

1. 各类试点班均采用网上报名的方式。学生凭姓名及高考考生号（或身份证号）登录武汉理工大学教务处网站http://jwc.whut.edu.cn/，进入“试点班报名系统”报名。

2. 报名时间：2020年9月5日—2020年9月15日。

3. 学生自愿报名，每种选拔类型最多填报1个志愿（可放弃填报某种类型试点班的志愿）。

（五）录取

学校按批次进行试点班录取。其中：“全校选拔的试点班”第一批录取，“学院内部选拔的试点班”第二批录取。学生填报的前一批次志愿被满足后，不能再进入后续批次的录取。请学生填报时务必慎重考虑。

例1：学生在“全校选拔的试点班”中报名信息与计算科学实验班（第一批录取专业），在“学院内部选拔的试点班”中报名交通学院船舶与海洋工程学硕连读班（第二批录取专业）。由于信息与计算科学实验班在第一批录取，学生若达到信息与计算科学实验班的录取要求，即被理学院信息与计算科学专业录取，不再进入后续交通学院船舶与海洋工程学硕连读班的录取环节。

例2：学生只准备进入“机电工程学院机械工程学硕连读班”学习，则应放弃填报“全校选拔的试点班”志愿，而在“院内选拔的试点班”中填报“机电工程学院机械工程学硕连读班”。

（六）特别提醒

1. 请学生及家长务必详细了解各类试点班情况后再填报志愿。学生一旦报名、参加了考核并被相应试点班录取，学籍即转到相应试点班对应的学院、专业。学校不再受理学生退出或转往其他试点班的申请。

2. 学生要退出试点班或转往其他专业，只能在学校每年4月份启动全校转专业工作时提出转专业（或退出）申请。

3.请学生在考试时务必填涂正确的学号。

（七）被限制参加试点班选拔学生

1. 以下各类学生，不能选拔到上述试点班：

⑴ 录取方式为定向生、高水平运动队运动员的；

⑵ 录取到国际教育学院的学生、艾克斯马赛学院的学生、艺术类专业（设计学类、动画专业）学生、航海类专业（航海技术、轮机工程专业）学生；

2. 以外国语言文学类专业保送的学生只能申请外国语学院英法双语实验班。

三、建筑类选拔考试

1. 按照我校招生要求，建筑类专业学生进校后需进行美术加试。2020级新生美术加试时间定于9月17日下午14:30。

2. 考试资格：2020级已经录取到建筑类专业的新生、第一志愿填报建筑类专业但未被录取的新生。
