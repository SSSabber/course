管理员 首页

构建教学资源共享平台的目的是，支持大学本科教学过程中教师和学生之间的信息共享和交流。该平台具有以下主要功能：
1）在Lucene的支持下，为ppt、word 、excel、pdf、html等格式的电子教学资料构建索引，对全文检索提供支持。
2）设计存储结构。在文件系统和Lucene的支持下，支持各类文档、索引、及其他相关信息的存储；引入数据库系统，支持结构化数据的存储。
3）支持学生按照导航、关键词和关键句的方式访问和下载平台上的教学资料。
4）支持教师上传、更新和维护教学资料。
5）支持用户角色和权限的管理，支持教学资料权限的设置和管理，支持用户群的设置和管理。
以此为基础，支持学生在特定的用户群中上传作业和实验报告等电子文档，支持教师在特定的用户群中批阅和下载。
基于对需求和业务规则的理解，在设计与实现阶段需要完成以下工作：
1）针对需求，选择设计与实现阶段采用的技术和方法。
2）数据结构和存储结构的设计。
3）程序功能结构和用户界面的设计。
4）程序的实现与调试。
